Highlights

This is the official 12.0.0 release, which is fully compatible with Angular 12.

To update to Clarity v12, first update to Angular 12 following their update guide. Then to update Clarity, run the update command below to have it update necessary dependencies. Finally, review any breaking changes that might impact you and update as necessary. 


ng update @clr/angular@latest
